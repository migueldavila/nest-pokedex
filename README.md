# Run on dev

1. Clone repo
2. Execute
```
npm install
```
3. You've got Nest CLI installed
```
npm install -g @nestjs/cli
```
4.Up and Run Database
```
docker-compose up -d
```
## Stack
* MongoDB
* Nest
